ARCADE BROWSER V4 — LOCAL MAC APP

This is not a website wrapper.

Bundled inside the app:
- Dashboard and navigation
- Admin Studio
- All 12 original games
- Themes, favorites, achievements, history
- Browser interface and app detection

Internet is used only for:
- Supabase accounts and cloud data
- Stripe payments
- approved advertising and memberships
- update checking
- websites the user chooses to open

The browser page opens websites in a separate Arcade Browser window.

NO-TERMINAL BUILD:
1. Create a GitHub repository.
2. Upload every file in this folder, including .github.
3. Open Actions.
4. Run "Build Local Arcade Browser Mac DMG".
5. Download the ArcadeBrowser-V4-Mac artifact.

LOCAL MAC BUILD:
Double-click BUILD-MAC.command.

The finished app is detected as Desktop App.
The final DMG is not automatically Apple-signed or notarized.
